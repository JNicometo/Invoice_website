Release Notes

What's new in OwnInvoice Desktop. We ship monthly bug fixes and quarterly feature updates.

v1.0.0 Latest March 2026

Features

  • Full Desktop Application — Native app for Windows and macOS, runs completely offline
  • Professional Invoicing — Create, edit, and manage invoices with customizable line items, discounts, tax, shipping, and adjustments
  • Quotes & Estimates — Create proposals and convert accepted quotes directly into invoices
  • Credit Notes — Issue refunds and billing corrections linked to specific invoices
  • Client Management — Unlimited clients with contact details, multiple addresses, credit limits, and payment terms
  • Saved Items Library — Reusable product and service catalog with categories, SKUs, and markup calculation
  • Recurring Invoices — Automate weekly, bi-weekly, monthly, quarterly, or yearly billing
  • Payment Reminders — Automated and manual email reminders for overdue invoices with customizable templates
  • Reports & Analytics — Revenue tracking, monthly charts, client revenue breakdown, and CSV export
  • 5 Payment Gateways — Stripe, PayPal, Square, GoCardless, and Authorize.Net integration with sandbox mode
  • Email Integration — Send invoices and reminders via SMTP with PDF attachments
  • PDF Export — Professional PDF generation with customizable templates, fonts, colors, and layouts
  • Batch Operations — Select multiple invoices for bulk mark-as-paid, archive, or delete
  • Dashboard — Business overview with revenue stats, pending/overdue amounts, and recent invoices
  • Global Search — Search across invoices, clients, and items with keyboard shortcut
  • Backup & Restore — Automatic scheduled backups with configurable retention and manual backup option
  • Archive System — Move old invoices and quotes out of active views with restore capability
  • CSV Import — Bulk import clients and saved items from CSV files
  • Customizable Interface — Dark mode, color themes, configurable sidebar, and keyboard shortcuts
  • Multiple Database Support — SQLite (default), MySQL, PostgreSQL, and SQL Server
  • License Activation — Secure license key system with machine-based activation
